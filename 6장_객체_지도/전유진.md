# 객체 지도

"유일하게 변하지 않는 것은 모든 것이 변한다는 사실뿐이다." - 헤라클레이토스 -

## 길찾기

- 길묻기 VS 지도
  = 기능적이고 해결방법 지향적 접근 VS 구조적이고 문제 지향적 접근법

* 지도의 범용성

- 지도는 안정적인 실세계 지형을 기반으로 한추상 모델
- 안정적인 구조에 변경이 발생하는 기능을 종속시키는 지도
  - 객체 지향
  * 범용적
  * 높은 재사용성
  * 변경에 안정적

## 기능 VS 구조

- 요구사항의 지속적 변화
  => 기능 + 예측 불가 요구사항 변경에 유연한 대처로 안정적 구조 제공
